# d3-topojson-map
This demo demonstrates how to generate a D3 map from a TopoJSON data source.

## Running the app
* Clone the repository into a directory, call it 'x'
* 'cd x'
* python -m http.server 8888 --bind localhost
* In a web browser:
  * http://localhost:8888
